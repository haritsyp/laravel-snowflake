# Haritsyp\Snowflake

[![Latest Version on Packagist][ico-version]][link-packagist]
[![Software License][ico-license]](LICENSE.md)

Snowflake ID generator for Laravel, supports int64 & Base62.

## Install 

In your terminal:

``` bash
# install the package
composer require haritsyp/laravel-snowflake
```
